# Glossary

## Terms

| Term | Definition |
| --- | --- |
| OWASP | Open Web Application Security Project |
| CWE | Common Weakness Enumeration |
| XSS | Cross-Site Scripting |
| SQLi | SQL Injection |
| Agent Shield | Prompt injection protection |
| Completion Guard | Mutation detection for completions |
| Five-Axis Review | Multi-perspective code review |

| Broken Auth | A02:2021 | Authentication failures |
| Sensitive Data | A02:2021 | Data exposure |
| XXE | A05:2021 | XML external entity |
| Broken Access | A01:2017 | Authorization failures |
| Misconfiguration | A05:2021 | Security misconfiguration |
| XSS | A03:2021 | Cross-site scripting |
| Insecure Deser | A08:2021 | Insecure deserialization |
| Components | A06:2021 | Vulnerable components |
| Insufficient Logging | A09:2021 | Logging failures |
