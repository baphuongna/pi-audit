# pi-audit

Extension for Pi coding agent.

## Install

### From npm (recommended)

```bash
pi install npm:pi-audit
```

### From GitHub Packages

```bash
pi install npm:@baphuongna/pi-audit
```

### From local source (development)

```bash
cd pi-audit && pi install .
```

## Verify

```bash
pi list
```
